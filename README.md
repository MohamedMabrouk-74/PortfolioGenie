## 🎓 About

> Graduation Project – **Digital Egypt Pioneers Initiative (DEPI)** 🇪🇬

**PortfolioGenie** is a team-built, AI-powered platform that transforms GitHub activity into polished, recruiter-ready portfolios.

## 🧠 Problem → Solution → Impact

### ❌ Problem

* Junior developers struggle to present projects professionally.

### ✅ Solution

* Analyze GitHub activity + generate optimized content using AI.

### 🚀 Impact

* Faster portfolio creation
* Better recruiter visibility
* Stronger personal branding

---

## ✨ Features

* 🤖 AI-generated project descriptions (SEO optimized)
* 🧠 Smart "About Me" suggestions
* ⚛️ Built with React Hooks
* 🎨 Dark UI design
* ⚡ High performance with Vite
* 📱 Responsive design

---

## 🧩 Architecture

```text
Client (React + Vite)
  ├─ Pages
  ├─ Components
  ├─ Hooks
  └─ Styles

AI Layer
  └─ Prompt Engineering

Data Layer
  └─ GitHub API
```
